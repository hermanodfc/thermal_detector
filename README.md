# Thermal Detector

Um aplicativo feito em Flutter que permite a comunicação entre Smartphones e sensores MLX90640 acoplados em uma placa ESP32.  

### Requisitos:

- Placa ESP32 (ou compatível) com bluetooth;
- Sensor MLX90640.

### Características:   

- Permite realizar a construção de uma imagem térmica a partir dos dados capturados, os quais serão salvos para uso posterior;
- Elimina a necessidade de compilação do software embarcado quando for necessário fazer ajustes nos parâmetros de configuração do sensor.
